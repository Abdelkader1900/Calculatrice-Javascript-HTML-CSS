# Calculatrice Quantique Pro Max

Calculatrice interactive développée en HTML, CSS et JavaScript vanilla, permettant d'effectuer les 4 opérations de base.

## Fonctionnalités

- Addition, soustraction, multiplication, division
- Bouton C pour réinitialiser le calcul
- Interface responsive et stylisée

## Technologies

- HTML
- CSS
- JavaScript

## Lancer le projet

Ouvrir `index.html` dans un navigateur.
